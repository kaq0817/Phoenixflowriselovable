# AI Generator Instructions for Shopify & Etsy Optimization


## Shopify Product Optimization

- **Title:** Descriptor-first, under 70 chars, no vendor/brand names. Format: [Descriptor] [Item Type] [Key Attribute if critical]. No store/vendor names. Example: "Block World Pixelated Travel Mug".
- **SEO Title:** Under 60 chars. Append store name only if result stays under 60 chars. No vendor names.
- **Meta Title:** Max 60 chars, keyword-focused.
- **Meta Description:** 120-155 chars, clear, no promo fluff, must include primary keyword.
- **Description (body_html):** Use <h3>, <p>, <ul>, <li>, <strong> only. H3 headings (Features, Benefits, Specs). One bullet list (3-5 items). Factual, no exaggerated claims.
- **Tags:**
  - 5 to 15 tags per product is the industry standard for clear navigation and efficient collections.
  - 3 to 5 tags per product if you want to strictly avoid tag sprawl and thin content.
  - 250 tags is the absolute technical maximum per product (Shopify limit; never exceed).
  - Tags must be short (under 16 characters), use only letters, numbers, and hyphens (no special characters like & or _), and be consistent (prefer singular form, e.g., "Shirt" not "Shirts").
  - Think like a real shopper. Use buyer-intent search phrases for the niche. Preserve all specific tags. Upgrade generic tags with themed niche terms. No vendor names. Each tag max 255 chars.
- **URL Handle:** Hyphenated, lowercase, keyword-based, max 60 chars.
- **FAQ:** JSON array string of 3-4 Q&A pairs.
- **Image Alt Text:** Every image must have descriptive, SEO-friendly alt text under 125 chars. Format: "[Product Name] - [Detail/Angle] | [Store]". No generic text, no vendor names.
- **Image Filenames:** Lowercase, hyphen-separated, no special chars, end in .jpg. Format: "[clean-product-name]-[detail]-[store].jpg". No generic names, no vendor names.
- **Compliance:**
  - Apparel titles must include color and size range.
  - No special characters, Unicode, emojis, or ALL CAPS (except acronyms).
  - No promotional text ("FREE SHIPPING", "SALE").
  - No medical/health claims (Facebook/Meta compliance).
  - Descriptions must describe what the product IS, not what it DOES to the body.

## Etsy Listing Optimization

- **Title:** Max 140 chars, aim for under 15 words. Most important traits first. Concise, readable, no keyword cramming.
- **Tags:** Exactly 13, each max 20 chars. Prefer multi-word, real search phrases. No generic one-word tags unless required. No repeats or near-duplicates. No wasted tags repeating title words unless stronger long-tail. Apparel: color must be a tag.
- **Description:** Informative, engaging, strong first sentence stating what the item is. Essential details at the top. Use keywords naturally, no title copy-paste. Short paragraphs or bullet lists for readability.
- **Materials:** Accurate, specific list.
- **Compliance:**
  - No special characters, Unicode, emojis, or ALL CAPS (except acronyms).
  - No promotional text ("FREE SHIPPING", "SALE").
  - No excessive punctuation.
  - Descriptions must be factual, no exaggerated claims.

## General Rules

- Use keywords naturally, never stuff or force them.
- Always match search intent (commercial/transactional for products).
- Never use vendor/store names in titles, tags, or alt text.
- All content must be unique, clear, and conversion-focused.
- Any violation of these rules is a hard SEO fail.
